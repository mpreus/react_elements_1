Siplified shopping cart made in react with one item in it and specified number of such items in stock.
You can mark how many items you like to buy and click BUY NOW button. That move updates the stock.
During buying process, you can also resign. When number of purchasing items is 0, '-' button is inactive, as well as the buton '+' when number of buying items reaches available number of items.
The stock is updated dynamically.
